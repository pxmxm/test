# 管理服务机构上传资质资料

## 请求

#### 请求方法： `get`

#### 请求地址： `http://{{ip}}:{{port}}/api/reset_password/?username=<统一性用代码>`

#### 请求头：

```json
{
  "Content-Type": "application/json"
}
```

#### 查询参数

* 无

#### 路径参数

| 字段               | 类型   | 必须 | 说明                           | 默认 |
| ------------------ | ------ | ---- | ------------------------------ | ---- |
| `username`             | String | 是   | 统一性用代码                       |      |


#### 请求体参数

| 字段               | 类型   | 必须 | 说明                           | 默认 |
| ------------------ | ------ | ---- | ------------------------------ | ---- |
