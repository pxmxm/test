# 管理服务机构上传资质资料

## 请求

#### 请求方法： `post`

#### 请求地址： `http://{{ip}}:{{port}}/api/reset_password/`

#### 请求头：

```json
{
  "Content-Type": "application/json"
}
```

#### 查询参数

* 无

#### 路径参数

| 字段               | 类型   | 必须 | 说明                           | 默认 |
| ------------------ | ------ | ---- | ------------------------------ | ---- |


#### 请求体参数

| 字段               | 类型   | 必须 | 说明                           | 默认 |
| ------------------ | ------ | ---- | ------------------------------ | ---- |
| `name`             | String | 是   | 单位名称                       |      |
| `username`             | String | 是   | 统一性用代码                       |      |
| `mobile`             | String | 是   | 联系电话                       |      |
| `smsCode`             | String | 是   | 验证码                       |      |
| `password`             | String | 是   | 密码                       |      |
| `password2`             | String | 是   | 确认密码                       |      |
