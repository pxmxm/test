## 请求

#### 请求方法： `get`

#### 请求地址： `http://{{ip}}:{{port}}/api/experts/expert_query/`

#### 请求头：

```json
{
  "Content-Type": "application/json"
}
```

* 无

| `idCardNo`             | String | 否   | 身份证                       |      |
