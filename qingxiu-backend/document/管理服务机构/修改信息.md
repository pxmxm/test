# 管理服务机构上传资质资料

## 请求

#### 请求方法： `put`

#### 请求地址： `http://{{ip}}:{{port}}/api/agency/modify/`

#### 请求头：

```json
{
  "Content-Type": "application/json"
}
```

#### 查询参数

* 无

#### 路径参数

| 字段               | 类型   | 必须 | 说明                           | 默认 |
| ------------------ | ------ | ---- | ------------------------------ | ---- |


#### 请求体参数

| 字段               | 类型   | 必须 | 说明                           | 默认 |
| ------------------ | ------ | ---- | ------------------------------ | ---- |
|id|string|是|评估机构ID|
|mobile|string|是|联系电话|
|contact|string|是|联系人|
|qualification|list|是|资质资料|
|smsCode|string|否|验证码| 不修改手机号时验证码可以为空


## 响应

#### 响应数据

#### 错误代码

| 错误代码 | 说明             |
| -------- | ---------------- |
|1|请输入正确的联系电话
|1|请输入手机验证码
|1|验证码失效，请重新获取验证码'}
|1|验证码错误


## 示例

#### 请求

`put http://{{ip}}:{{port}}/api/agency/modify/`
```json
15108050981
```

#### 响应

```json
{
	"code": 0,
	"message": "修改成功"
}
```
