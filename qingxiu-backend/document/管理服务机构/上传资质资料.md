# 管理服务机构上传资质资料

## 请求

#### 请求方法： `post`

#### 请求地址： `http://{{ip}}:{{port}}/api/agency_register/upload/`

#### 请求头：

```json
{
  "Content-Type": "application/json"
}
```

#### 查询参数

* 无

| 字段               | 类型   | 必须 | 说明                           | 默认 |
| ------------------ | ------ | ---- | ------------------------------ | ---- |


#### 请求体参数

| 字段               | 类型   | 必须 | 说明                           | 默认 |
| ------------------ | ------ | ---- | ------------------------------ | ---- |
| `file`             | file | 是   | 文件                       |      |
| `fileName`             | String | 是   | 文件名称                       |      |
