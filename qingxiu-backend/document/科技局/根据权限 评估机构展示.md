# 管理员查询拥有立项的评估机构

## 请求

#### 请求方法： `get`

#### 请求地址： `http://{{ip}}:{{port}}/api/agency/agency_show/?permissions=1`

#### 请求头：

```json
{
  "Content-Type": "application/json"
}
```

#### 查询参数

* 无

#### 路径参数

| 字段               | 类型   | 必须 | 说明                           | 默认 |
| ------------------ | ------ | ---- | ------------------------------ | ---- |
|permissions|string|是|{（1，立项评估管理）（2，结题终止管理）

#### 请求体参数

| 字段               | 类型   | 必须 | 说明                           | 默认 |
| ------------------ | ------ | ---- | ------------------------------ | ---- |
